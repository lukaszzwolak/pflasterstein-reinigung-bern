# Pflasterstein Reinigung Bern – Landing Page

A clean, modern, SEO-optimized landing page for a **pavement / stone cleaning service (Pflasterstein Reinigung)** in **Bern, Switzerland**.  
The project is built both as a **real business idea** and as a **portfolio project** for web development.

## Features

- Hero section with CTA: **“Kostenlose Offerte anfragen”** (Request free quote)
- Service overview:
  - High-pressure cleaning
  - Moss & algae removal
  - Terrace & driveway cleaning
  - Stone impregnation
- Price guidelines (CHF per m²)
- Before/After section (initial placeholders, replace with real images later)
- Interactive price calculator (m² + service type + impregnation option)
- Contact form
- WhatsApp, email and phone quick contact links
- Local SEO optimized for **Bern / Canton Bern** (meta tags, keywords, location section)
- Fully responsive layout (mobile, tablet, desktop)

## Tech Stack

**Frontend only (no backend required):**

- **HTML5** – Page structure & content
- **CSS3** – Custom responsive layout, modern “Swiss-clean” design
- **JavaScript (Vanilla ES6)** – Scroll behavior, price calculator, form handling

The project is deployed as a **static site**.
